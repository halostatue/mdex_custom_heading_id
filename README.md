An MDEx plugin that supports custom header IDs

## Installation

Add `mdex_custom_heading_id` to your dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:mdex_custom_heading_id, "~> 1.0"}
  ]
end
```

Documentation is found on [HexDocs][docs].
